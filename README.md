# 🚀 VibeLogs\n\nLa bitácora de trabajo del futuro, hoy.\n\nAplicación SaaS construida con Next.js y Supabase para gestión de bitácoras de productividad.\n\n## Tech Stack\n- **Frontend**: Next.js (App Router), React, TypeScript\n- **Backend**: Supabase (Auth, Database, Storage, Edge Functions)\n- **Styling**: CSS con Glassmorphism + styled-jsx\n- **PDF**: jsPDF + jspdf-autotable\n\n## Setup\n\n```bash\nnpm install\nnpm run dev\n```\n\nConfigure las variables de entorno en `.env.local`:\n```\nNEXT_PUBLIC_SUPABASE_URL=your_supabase_url\nNEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key\n```\n
